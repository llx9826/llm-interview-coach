# Demo Script

这个文件用于录屏、做 gif、做截图，尽量让展示更像真实使用，而不是静态文档。

## 场景 1：项目深挖

输入：

```text
用 llm-interview-coach 深挖我的项目：
我做了一个企业知识库问答系统，采用 hybrid retrieval + rerank + generator，
线上平均延迟 1.2s，回答采纳率提升 11%，我主要负责检索、评测和服务优化。
```

展示重点：

- 一上来先给短结论
- 不直接给标准答案
- 问题围绕 baseline、ownership、失败方案展开
- 最后给诊断和训练动作

## 场景 2：个性化训练

输入：

```text
用 llm-interview-coach 按我的背景做个性化训练：
- 3 年算法工程经验
- 做过 RAG、评测和推理服务
- 弱项是 RLHF、训练细节、英文表达
- 目标是 2 周后面中文大厂大模型算法岗
```

展示重点：

- 输出候选人画像
- 给出短板 Top 5
- 给出 7 天或 14 天训练顺序

## 场景 3：Mock interview

输入：

```text
用 llm-interview-coach 帮我做一轮中文大厂大模型算法岗 mock，重点考 RAG、LoRA 和推理优化
```

展示重点：

- 一次只问一道题
- 回答后继续追问
- 用诊断模板收尾

## 录屏建议

- 用等宽字体终端
- 保持 30-45 秒一段，不要过长
- 每段只展示一个核心卖点
- 如果做 gif，优先录“项目深挖”场景
