## GitHub Post / Reddit / V2EX 长版

```text
I open-sourced `llm-interview-coach`, a Codex skill for LLM interview prep.

The goal is simple: most interview prep material is either too generic or too disconnected from real project experience. This skill tries to fix that by focusing on:

- project-first interrogation
- role-aware question selection
- realistic follow-up pressure
- structured diagnosis after each round
- personalization based on the candidate's own background

It is especially tuned for Chinese-market LLM interviews, but the overall workflow should also be useful for broader GenAI interview prep.

Repo:
https://github.com/llx9826/llm-interview-coach
```
